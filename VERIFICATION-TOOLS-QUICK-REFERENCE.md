# ⚠️ Integrated Verification Tools - Quick Reference

## 你看到的警告是什么？

```
⚠️ Integrated Verification Tools
在使用这个输出前，强烈建议先验证一下关键内容。
我已准备了验证工具供你使用。
```

这不是错误！这是系统提醒你：**有验证工具可以帮你检查 AI 的答案**。

---

## 🔍 如何使用？

### 方法 1️⃣: 点击聊天输入框旁的 🔍 Verify 按钮

在聊天界面底部，你会看到：
```
┌─────────────────────────────────┬──────┬──────┐
│ Ask a question...               │ Verify │ Send │
└─────────────────────────────────┴──────┴──────┘
                                    🔍    📤
```

**直接点击 🔍 Verify 按钮** → 验证工具窗口会打开

### 方法 2️⃣: 看到自动提醒时

当系统检测到你需要验证时，会显示警告。点击提醒中的验证工具链接。

---

## 🚀 快速使用（3步）

### Step 1: 打开验证工具
- 点击聊天输入框旁的 **🔍 Verify** 按钮
- 会看到一个大窗口，顶部有 3 个标签：
  - ✅ **Verify Content** (验证内容) - 主要功能
  - 📋 **Verification History** (验证历史) - 你的历史记录
  - 📊 **Verification Stats** (验证统计) - 统计数据

### Step 2: 选择内容类型并输入

输入你想验证的内容（从 AI 复制）：

| 类型 | 用 | 验证方法 |
|------|-------|--------|
| **💻 Code** | AI 的代码 | 运行代码 / 检查语法 |
| **🔢 Math** | 数学公式 | 计算验证 / 交叉检查 |
| **📖 Citation** | 引用文献 | Google Scholar / 检查来源 |
| **📰 Fact** | 事实声明 | Fact-check API / 检查权威源 |
| **📝 Text** | 文本/文章 | 所有方法都可用 |

### Step 3: 选择验证方法并查看结果

系统会推荐最适合的方法。选择后它会：
- ✅ 检查内容
- ⚠️ 告诉你哪里有问题
- 💡 给出改进建议

## 📚 具体例子

### 例子 1：验证代码

```
你: "写一个反转字符串的 Python 函数"
AI: "def reverse(s): return s[::-1]"

1. 点击 🔍 Verify
2. 复制粘贴代码到文本框
3. 选择类型: 💻 Code
4. 点击 "Continue"
5. 选择验证方法:
   - ⚙️ Code Execution (运行代码测试)
   - ✓ Syntax Check (检查语法)
6. 点击 "Code Execution"
7. 看到结果:
   - Test: reverse("hello") → "olleh" ✅
   - Test: reverse("a") → "a" ✅
   - Test: reverse("") → "" ✅
8. 决定: ✅ Accept (接受，可以使用)
```

### 例子 2：验证数学

```
你: "计算 2^10"
AI: "2^10 = 1024"

1. 点击 🔍 Verify
2. 输入: "2^10 = 1024"
3. 选择类型: 🔢 Math
4. 选择方法: 🧮 Calculation
5. 结果: ✅ Correct!
6. 决定: ✅ Accept
```

### 例子 3：验证事实

```
你: "中国的首都是什么？"
AI: "中国的首都是北京"

1. 点击 🔍 Verify
2. 输入: "中国的首都是北京"
3. 选择类型: 📰 Fact
4. 选择方法: ✔️ Fact-check
5. 结果: ✅ Verified! Beijing is the capital of China
6. 决定: ✅ Accept
```

## ✅ 验证后你可以做什么？

看到验证结果后，你有 4 个选择：

```
1. ✅ Accept (接受)
   → 这个内容是对的，我要用它

2. ❌ Reject (拒绝)
   → 这个内容有错，我不用它

3. 🔄 Revise (修改)
   → 让 AI 修复错误，然后再验证

4. 📝 Note (备注)
   → 记录我的发现和决定
```

系统会：
- 💾 保存你的选择
- 📊 统计你的验证模式
- 📈 追踪你的改进

## 🎯 为什么要验证？

### 问题：
```
AI 答案 → 你直接复制 → 错误被忽略 → 你没学到东西
```

### 解决：
```
AI 答案 → 你验证 → 找到错误 → 你理解为什么 → 真正学到
```

### 好处：
- 🎓 **学得更好** - 理解每个答案
- 🛡️ **避免错误** - 在使用前就发现问题
- 📈 **看到进展** - 追踪你的改进
- 🧠 **独立思考** - 不依赖 AI 的自动答案

## 🔧 常见问题

### Q: 验证结果准确吗？
**A:** 取决于内容类型
- 💻 代码: 很准确（能实际运行）
- 🔢 数学: 中等准确（取决于复杂性）
- 📖 引用: 中等准确（取决于数据库）
- 📰 事实: 可能不太准确（取决于信息新旧）

**建议**: 对重要的内容，用多个方法验证

### Q: 验证会花多长时间？
**A:** 通常 1-5 秒，取决于内容类型

### Q: 我可以离线验证吗？
**A:** 不可以，验证需要网络

### Q: 可以验证的最大内容大小是多少？
**A:** 通常 100KB，可以粘贴很长的代码或文本

### Q: 验证历史会永久保存吗？
**A:** 是的，直到你清除它

### Q: 我可以看到其他用户的验证结果吗？
**A:** 不可以，验证是私密的

## 🔐 隐私和数据

### 验证工具：
- ✅ **不会** 分享你的验证内容给其他用户
- ✅ **不会** 用你的内容训练 AI
- ✅ **会** 保存你的验证历史（只有你看得到）
- ✅ **会** 使用统计来改进验证功能

### 你的选择：
- 📝 随时可以删除验证历史
- 🔒 验证是完全私密的
- 🚫 可以选择不使用验证工具
