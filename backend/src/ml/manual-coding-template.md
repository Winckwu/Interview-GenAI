## 📝 质量检查清单

### 内部一致性检查

- [ ] P1高但P2低？合理吗？（任务分解但目标不清晰可能合理）
- [ ] M2高但E1低？合理吗？（实时检查但最终不评估可能不合理）
- [ ] R1高但M2低？合理吗？（迭代调整但不验证可能不合理）
- [ ] P4高但E3低？合理吗？（定义边界但不判断能力可能不合理）

### 与同模式其他用户对比

- [ ] 总分是否在该模式的预期范围内？
- [ ] 关键子过程分数是否与模式特征一致？
- [ ] 是否有明显偏离模式的子过程？如果有，为什么？

### 证据质量检查

- [ ] 每个分数都有具体引述支持？
- [ ] 引述是否明确展示行为（而非仅意识）？
- [ ] 是否区分了"知道"vs"做到"？
- [ ] 是否有至少2个示例支持强证据（✓✓✓）？
