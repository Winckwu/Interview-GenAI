## 1. 转录规范

### 1.1 转录符号说明

| 符号 | 含义 |
|------|------|
| Line X: | 原始访谈行号引用 |
| **粗体** | 编码关键片段 |
| `代码` | 开放编码标注 |
| ✓✓✓ | 证据强度（3分） |

### 1.2 编码层级

```
开放编码 → 主轴编码 → 选择性编码 → 理论整合
   │           │            │            │
   ↓           ↓            ↓            ↓
 概念化      类别化       核心范畴      六模式框架
```
